# 🚀 Idata 

**iData** is an open-source mockup API with enormous datasets integration for even beginner developers. For a better understanding of our topic, Let us first understand the concept of an API.



**🎯 Project Overview** This system provides a centralized workspace to manage API Schemes, Folders, and Project Analytics. It leverages Next.js 16, Redux Toolkit, and Tailwind CSS to ensure a high-performance, scalable experience for full-stack developers.

## Installation

1. Download/fork/clone the repo and Once you're in the correct directory, it's time to install all the necessary dependencies. You can do this by typing the following command:

```
git clone https://github.com/kimla1234/final-project-idata-ui
```
make file .env.development
```
NEXTAUTH_SECRET=98E3B2CC28F61492C6934531C828C
NEXTAUTH_URL=http://localhost:3000

NEXT_PUBLIC_BASE_URL_LOCALHOST=http://localhost:3000/api/

COOKIE_REFRESH_TOKEN_NAME=final-refresh-token

NEXT_PUBLIC_NORMPLOV_API_URL=http://localhost:8081

```
make file .env.production for production
```
NEXTAUTH_SECRET=98E3B2CC28F61492C6934531C828C
NEXTAUTH_URL=http://localhost:3000

# NEXT_PUBLIC_BASE_URL_LOCALHOST=http://localhost:3000

COOKIE_REFRESH_TOKEN_NAME=final-refresh-token

NEXT_PUBLIC_NORMPLOV_API_URL=https://api.idata.fit

```

2. Install package 
```
npm install 
```

3. Okay, you're almost there. Now all you need to do is start the development server. If you're using **npm**, the command is:

```
npm run dev
```
And if you're using **Yarn**, it's:

```
yarn dev
```



And voila! You're now ready to start developing. **Happy coding**!
